# Como usar

## 1 passo
### clonar repositório
```bash
git clone https://github.com/henriquepython/nestwithgraph.git
```

```bash
## entrar pasta projeto
cd nestwithgraph
```


## 2 passo

```bash
# rodar banco
docker-compose up

# abra outro terminal

# instalar dependências
npm install

# executar o projeto
nest start

# Acesso ao playground
http://localhost:3000/graphql
```
